# SDVer1 - Login Page

This is a GitHub-ready Vite + React project containing the Login Page for SDVer1.

## How to use

1. Unzip and open project folder
2. Install dependencies:
   ```
   npm install
   ```
3. Run development server:
   ```
   npm run dev
   ```
4. Build for production:
   ```
   npm run build
   ```
5. Deploy to GitHub Pages (after adding remote):
   ```
   npm run deploy
   ```

Note: `vite.config.js` is configured with `base: '/SDVer1/'` for GitHub Pages.
